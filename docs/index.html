How It Works

1

Scan

Analyze your codebase with Tree-sitter parsing to build a complete dependency graph and identify complexity hotspots.

2

Plan

LLM-powered agents generate detailed refactoring plans with diffs, showing exactly what will change and why.

3

Apply

Execute changes safely with automatic test validation, git checkpoints, and instant rollback on failure.

üîß Build from Source

Requires Go 1.21+ and Python 3.10+:

# Clone the repository
git clone https://github.com/alexkarsten/reducto.git
cd reducto

# Install Python dependencies
pip install -r python/requirements.txt

# Build the binary
go build -o reducto ./cmd/reducto
